# skills

> Multi-harness skills & agents for Claude Code, Codex, and Gemini CLI.

A curated, installable library of skills and agents that works across multiple AI coding harnesses. Pick what you want, install via script or `npx`, and the installer symlinks each selected item into the right location for your harness.

## Status

🚧 **Early scaffolding.** The repository structure, installer, and initial skill set are still being designed. Star or watch the repo if you'd like to follow along.

## Goals

- **Multi-harness by default** — every skill/agent ships in a form that works on Claude Code, Codex, and Gemini CLI (where applicable).
- **Pick-and-choose installer** — users select which skills/agents to install rather than getting everything.
- **Symlink-based** — installed items stay in sync with the repo so `git pull` updates them in place.
- **NPX-runnable** — `npx @11-mile-labs/skills install` (or equivalent) without a global install.
